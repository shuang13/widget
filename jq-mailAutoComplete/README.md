# mailAutoComplete.js 
mailAutoComplete是一个自动补全邮箱后缀域名的jq插件
## Options参数
    mailAutoComplete({
	//参数名： 默认值
    	emailType: [
            '',
            '@qq.com',
            '@163.com',
            '@126.com',
            '@gmail.com',
            '@sina.com',
            '@sohu.com',
            '@outlook.com',
            '@139.com'
        ],
        width: 250,
        height: 200,
        index: 99,
        classname: 'selector-box',
    });
